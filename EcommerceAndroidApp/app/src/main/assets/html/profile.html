데이팜(DayFarm) - 농산물 직거래 장터


안녕하세요. 저희 데이팜은 대학생들로 이루어진 창업팀 '을의이야기'의 첫번째 서비스입니다. '을의이야기'는 항상 갑(?)님들의 그늘에 살고있는 저희가 한번 우리들이 하고싶은것, 하고자하는것을 자유롭게 해보자! 라는 생각으로 시작하게 되었습니다.
저희의 첫번째 서비스 데이팜 - 농산물 직거래 장터는 농촌과 고객을 직접 이어주는 소셜커머스 애플리케이션 입니다. 농어촌을 대상으로 하는 저희 데이팜 서비스는 일반적으로 물건을 사고파는 소셜커머스와 다른 차이점을 가지고 있습니다.


1. 농장을 대상으로한 무료 브랜딩 서비스
2. 농장의 수익을 극대화 할 수 있는 B급몰 서비스
3. 고객들의 직접 투자로 이어지는 크라우드 펀딩


기존의 것과 차별화된 서비스를 안고 저희 '을'들의 첫번째 이야기가 지금 시작합니다.


저희 '을의이야기' 데이팜 - 농산물 직거래 장터는 2015년 창업성장기술개발사업 - 이공계꿈나무 의 지원을 받아 수행하였습니다.
